¿Qué es Post-It-Note-SEAA?

Se trata de una plataforma donde se crean de manera virtual las notas, ya que en de manera física

es una pequeña hoja de papel con una tira de pegamento que se puede volver a adherir en su parte

posterior, hecha para adjuntar notas temporalmente a documentos y otras superficies.

¿Como funciona Post-It-Note-SEAA?

Para poder utilizar Post-It-Note-SEAA, primero se requiere que se registre para que se pueda crear

cuenta, una vez que se creo la cuenta inicia sesión y puede empezar por crea una categoría para

poder organizar las notas con mayor facilidad.

Para crear una clasificación solo se necesita resgistrar el nombre de la categoría y para poder crear

una nota se le da un titulo, se ingresa el contenido y la fecha de creación, esta última es para llevar

un control del registro y si son datos importantes eso será de mucha ayuda.
